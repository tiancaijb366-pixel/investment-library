# Prompt Engineering: The 3-Part Formula That Actually Works

After curating 100 ChatGPT prompts for developers, here's the one formula that consistently produces the best results:

## The Role + Task + Format Formula

Every effective prompt needs three parts:

### 1. Role
Tell ChatGPT who it is. This activates the right knowledge:
```
You are a senior full-stack developer with 15 years of experience...
```

### 2. Task
Be specific about what you want:
```
...review this React component for performance issues...
```

### 3. Format
Specify how to output:
```
...list issues as bullet points, ordered by severity, with code examples for each.
```

## Example

❌ **Bad:** "Review my code"
✅ **Good:** "You are a senior security engineer. Review this authentication code for vulnerabilities. List each issue with: (1) the vulnerable line, (2) why it's risky, (3) how to fix it."

## Why It Works
- **Role** sets context and expertise level
- **Task** removes ambiguity
- **Format** ensures the output is usable immediately
