# How to Use ChatGPT for Code Review — 10 Practical Prompts

Here are 10 ChatGPT prompts for effective code review, taken from our prompt pack.

## 1. General Code Review
"Review this code for bugs, performance issues, and maintainability concerns. Be specific about what should change."

## 2. Security Review
"Analyze this code for security vulnerabilities including injection attacks, authentication issues, and data exposure."

## 3. Performance Analysis
"Find performance bottlenecks in this code. Suggest specific optimizations with before/after examples."

## 4. Architecture Review
"Evaluate this code's architecture. Is it following SOLID principles? Suggest improvements for scalability."

## 5. Testing Gaps
"Identify areas in this code that lack test coverage. Suggest test cases that should be added."

## 6. Readability
"How can this code be made more readable? Focus on naming, comments, and structure."

## 7. Error Handling
"Review error handling in this code. Are edge cases properly handled? Suggest improvements."

## 8. Dependency Check
"Are there any unnecessary dependencies or coupling in this code? Suggest ways to reduce coupling."

## 9. API Design
"Review this API for consistency, RESTful principles, and developer experience."

## 10. Migration Safety
"Review this migration code. What could go wrong? Suggest rollback strategies."
